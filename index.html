História dos Primeiros Computadores

(Voltar ao Inicio)

Embora desde a antiguidade já existam aparatos para cálculos (um conhecido exemplo é o ábaco), e o conceito de computador programável ter sido criado por Charles Babbage e o primeiro algoritmo ter sido escrito por Ada Lovelace no século 19, é na década de 1940 que surge a Primeira Geração dos computadores eletrônicos.
Entre eles estão o Harvard Mark I, que é considerado o primeiro computador; o Colossus, criado para Cripto Análises pela Alemanha em meados da Segunda Guerra Mundial e o ENIAC, criado pelos Estados Unidos também em meados da Segunda Guerra sendo ele mais automático que seu antecessor, o Mark I.

No intervalo entre os anos de 1959 a 1965 os computadores a válvula foram perdendo espaço para os computadores operados com transistores. Os novos dispositivos ficaram significativamente menores, sendo uma grande evolução para a época. Essa é a chamada Segunda Geração.

Já a Terceira Geração é marcada pelos circuitos integrados (chips), o que ocorreu entre 1965 a 1970. Essa tecnologia diminuiu ainda mais o tamanho dos computadores e aumentou a capacidade de processamento.
A Quarta Geração, que data de 1971 até os dias atuais, é conhecida pela melhoria dos chips e surgimento dos microprocessadores e veio evoluindo até as tecnologias que conhecemos hoje. A interface gráfica, a revolução do mouse, os computadores pessoais, notebooks e até os smartphones são progressos que caracterizam essa geração, também marcada pela popularização e disseminação do uso da tecnologia.

A Quinta Geração é representada pela IA, Criptomoedas, Metaverso e Blockchain. Essas tecnologias estão sendo introduzidas nos últimos anos e evoluindo vertiginosamente.

Diferença entre Client e Server

(Voltar ao Inicio)

A estrutura da web é basicamente composta pelo lado Cliente (Client) e lado Servidor (Server) que se comunicam através da internet (protocolo HTTP).
O cliente, ou usuário, fazendo uso de um dispositivo como um computador ou smartphone por exemplo, acessa um site que está hospedado em um servidor. Esse site é formado pelos elementos HTML, CSS, Javascript e outros arquivos como fotos ou vídeos. Esses elementos são organizados e configurados para serem visualizados em um navegador (browser).
O servidor armazena informações em bancos de dados, o usuário faz o requerimento dessas informações e o servidor retorna com essas informações para o cliente.
As linguagens que são executadas nos lados tanto do cliente quando do lado do servidor diferem entre si. O lado do servidor é chamado de back-end e o lado do cliente é chamado de front-end. Entre as linguagens back-end estão o PHP, Ruby, Java, C#, e o Javascript (que também é utilizado no front-end). Já as linguagens front-end, as que são executadas pelo navegador são as já citadas CSS, o Javascript e o HTML (que não é uma linguagem de programação em si, e sim uma linguagem de marcação de texto) .

Estrutura HTML: Tags, Textos e Links

(Voltar ao Inicio)

O acrônimo HTML, em inglês, HyperText Markup Language, significa Linguagem de Marcação de Hipertexto. A estrutura básica de um documento HTML é começa com a abertura da tag <html> e dentro dessa tag vão os elementos . <head> e <body> . Na tag head vão os elementos que carregam antes de exibir a página e na tag body> vão todos os elementos que irão aparecer na tela para o usuário. Dentro da tag body podem ser utilizadas muitas outras tags que possuem atributos que podem formatar e organizar seu texto, inserir imagens e links (externos ou dentro do próprio site), inserir campos de entrada de informações e muitas outras funcionalidades.

Atributos das Tags

Atributos são informações que passamos para a Tag para que ela se comporte de forma esperada. São exemplos de atributos:

id: identifica um elemento que pode ser manipulado no javascript
style: tag mais voltada para o CSS, estiliza o seu texto direto no HTML, alterando cor, fonte, tamanho, etc.
class: parecida com a tag id.
input: insere um campo de entrada de dados, podendo ser numerico, alfanumérico, ou até mesmo um seletor de cores.
img src: insere uma imagem, que pode vir da pasta do site ou até mesmo de outro lugar da internet

Textos

Existem as tags que são voltadas para o texto de um site. Algumas delas são:

<h1> a <h6> : indicam títulos e subtítulos, sendo h1 a de maior tamanho, h2 menor que a h1 e assim por diante. São usadas para indicar títulos e subtítulos.
<p> : indica o parágrafo, é onde vai o "corpo" do texto.
<br> : quebra de linha
<strong> : coloca o texto em negrito
<u> : sublinha o texto.
<i> : coloca o texto em itálico.
<mark> : marca o texto com cor amarela
<del> : insere um texto com uma parte deletada e uma nova parte inserida.

Links

(Voltar ao Inicio)

Os links servem para proporcionar a navegação, sendo dentro de um mesmo site ou externamente, inserindo links direcionados para outros sites.
A tag e o atributo que se refere aos links é indicada por <a href=" ">
